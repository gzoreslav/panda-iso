# panda-iso
Local competitions results catalog

## Digital Ocen Deploy

Go to Digital Ocean terminal
```
ssh root@188.166.47.232
```

Stop\Start panda-iso
```
stop websitename
start websitename
```

Switch to panda-api
```
screen -ls -- show all screens
screen -r <screen_id>
```
ctrl+a ctrl+d -- to exit from screen
